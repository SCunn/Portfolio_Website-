Msc Games & XR Project: Room Scale Unity VR Game with 3d Spatial Mapping and 100% Hand Tracking Gesture Control

Video Demo: The project demo presented here is a Quest 2 roomscale VR experience that showcases hand tracking & spatial mapping technologies. The game is a survival horror wave style shooter game where the player must defend their environment from a horde of attacking enemies. Using the spatial mapping technology allows players to dynamically turn their physical environment into a fully immersive virtual environment. This game uses 100% hand tracking and gesture control to interact with the game's environment, while also encouraging 6 degrees of freedom movement within a room-scale environment.

Interaction Design

MA Interactive Media Project

This project explores interactive potential of electrical field sensing by the means of Capacitive technology. It is because of the technologies low-cost and versatility that it is relatively easy to apply the technology into everyday objects, an area that can be explored by using off-the-shelf materials. It is the simplification of the technology that makes it inclusive to all and paves the way for exploring easy to apply prototyping tools. The goal of this project was to create an interactive audio visual installation that explores low tech solutions for prototyping with interactive technologies. This project used used serial communication between an Arduino Uno board and a PC, sending sensor data from the Arduino board, and receiving it on the Processing IDE on a computer. This project used the Arduino CapSense Library by Paul Badger, and the SerialCallResponse sketches by Tom Igoe and Scott Fitzgearld.
